# 集合的定义

  • 集合是由指定范围内的满足给定条件的所有对象聚集在一起构成的

    • 每一个对象称为这个集合的元素

  • ZFC 公理化集合

    • 外延公理+空集存在公理+无序对公理+并集公理+幂集公理+无穷公理+替换公理+正则公理+选择公理

    ZFC 的含义

    其中 C 指的是选择公理，其余八个合称为 ZF 公理

集合示例 
所有英文字母
所有小于100的正奇数
中国所有的残疾人
世界上所有的数学家5
某植物园的所有植物
